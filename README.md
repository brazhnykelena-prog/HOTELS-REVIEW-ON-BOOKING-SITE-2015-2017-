# HOTELS-REVIEW-ON-BOOKING-SITE-2015-2017-
End-to-end analysis of Booking.com hotel reviews: cleaned and structured large raw data, handled missing values and outliers, analyzed traveler patterns, hotel scores, and seasonal trends, visualized insights with Plotly, and deployed an interactive Streamlit dashboard for exploring hotels and review behavior.

data link is :
https://www.kaggle.com/datasets/jiashenliu/515k-hotel-reviews-data-in-europe
